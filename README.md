 Team Task Manager

A Flask-based Team Task Management System with user authentication, project management, and task assignment features. 
The application is deployed using Railway and uses MySQL as the database.

Features:

User Authentication

 User Signup
 User Login
 Session Management
 Logout System
 Role-based Access (Admin / Member)

 Dashboard

Total Users Count
Total Projects Count
Total Tasks Count
Completed Tasks Count

 Project Management
 
Create Projects
View All Projects
Set Project Deadlines

 Task Management

 Assign Tasks to Users
 Set Task Priority
 Update Task Status
 View All Tasks
